# Editable Contracts

Editable contracts front-end, built in React. 

Uses a REST API back-end <https://github.com/bclarkau/editable-contracts-api>

### Install Dependencies

`yarn install`

### Webpack

`yarn watch` to watch all src files and start auto-reload server at localhost:8080
`yarn build` to build to dist folder
`yarn prod` to run production build to dist folder
